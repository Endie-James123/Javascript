Math Methods
Math.ceil() round up a decimal to the nearest integer
E.g Math.ceil(11.04) = 12


Math.floor() round down a decimal to the nearest integer
E.g Math.floor(11.04) = 11

Math.random() returns a random number between 0 (inclusive), and 1 (exclusive).
You can also get a random number between 0 and other numbers
E.g to get a random number between 0 and 3
Math.random() * 3
it'll give you a random number between 0 and 3(exclusive)
